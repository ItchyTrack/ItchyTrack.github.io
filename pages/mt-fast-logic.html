MT Fast Logic

About MT Fast Logic

Fast Logic is a mod created to improve the logic-blocks execution system and tools in the game Scrap Mechanic. User creations are scanned to build an abstract representation of user logic, which is then transformed to apply simple optimizations. The resulting logic dataflow graph is executed independently from the game ticks, allowing much faster logic execution than the original game supports. The mod also enables coloring of logic-block outputs and edges, and provides a multitool that allowing users to more easily create and interconnect large-scale logic arrays.

This project was made in collaboration with one other person. I developed the logic gates, graph optimizations, and simulator. He worked on the multitool. We worked together to allow these pieces to cleanly interact as one system. The mod has been downloaded over 2,300 times.

The Big Idea

This project focuses on high performance within a highly constrained execution environment, so great care was taken to run the logic as fast as possible while fitting into Scrap Mechanic's game system. I employed numerous optimization techniques such as disabling inactive gates (minimal recompute), converting gates to more optimal simulated versions (instruction selection), statistically based input ordering (instruction reordering for early exit), and combining gates together (instruction merging).
