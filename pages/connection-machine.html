Connection Machine

About Connection Machine

Connection Machine is an open-source, cross-platform desktop application I created for designing and simulating very large digital circuits. It provides an intuitive visual interface and a powerful simulation engine that lets users build and experiment with complex designs in real time.

I conceived and launched the project independently at the end of my first semester at RPI, and it became an official RCOSRensselaer Center for Open Source Software project the following semester. Since then, I’ve led a team of 15 developers — 14 from RPI and one international contributor — to expand and refine the platform’s capabilities.

What It Does

Connection Machine is designed as a faster, more intuitive alternative to existing logic simulators. Written in C++ and accelerated with Vulkan, it supports the creation and simulation of very large circuits at exceptionally high speeds.

Cool Features

  • WASM Integration: Users can import WebAssembly (WASM) files that define how to generate circuit blocks, enabling dynamic block creation based on user-defined parameters.
  • Tensor Connect: A powerful tool for defining arbitrarily complex wiring patterns by repeatedly projecting connections across space.
  • Vulkan Acceleration: The engine uses Vulkan for rendering, allowing over one million objects on screen simultaneously. Circuit simulation is also being migrated to the GPU for even greater performance.
